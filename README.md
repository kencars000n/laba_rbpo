# Task Management Service

## Тема проекта
Система управления задачами с REST API для организации работы над проектами.

## Основные сущности
- **User** - пользователи системы
- **Project** - проекты для группировки задач
- **Task** - задачи с статусами и приоритетами
- **Tag** - теги для категоризации
- **Comment** - комментарии к задачам

## Операции сервиса

### CRUD операции
- Создание, чтение, обновление, удаление для всех сущностей

### Бизнес-операции
1. **Назначение задачи пользователю** - связывает задачу с исполнителем
2. **Добавление тегов к задаче** - категоризация задач
3. **Статистика по проекту** - аналитика задач проекта
4. **Поиск задач по фильтрам** - расширенный поиск с фильтрацией
5. **Копирование задачи с комментариями** - создание полной копии задачи

---
*Разработано на Spring Boot + PostgreSQL*
